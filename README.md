## About me 
This is my repo for STATS 220. 

A little about me:

- I am undertaking a degree in **BCom / BSc** majoring in *Information systems, Finance and Stats* :)
- I am taking STATS 220 because I wanted to learn more about programming languages and visualising data sets. 
- I am interested in learning about the **R language** and making effective *data-driven decision-making*!

## Why I'm Taking STATS 220 
1. To learn programming with **R**
2. To understand how to *visualise* data effectively
3. To apply stats knowledge to real-world **decision-making**
4. To build skills that are useful and can complement my Finance and Information systems majors.
